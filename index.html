Summary: Latitude Vs. X


The purpose of this analysis was to identify trends in weather patterns as they relate to latitude. To accomplish this analysis, data was pulled from The OpenWeather Map API to assemble a dataset on over 500 cities.Each point of interest, temperature, humidity, cloudiness and wind speeed, was then averaged and plotted versus latitude for analysis. This site provides the source data and visualization created as part of the analysis and it also explains and describes any trends and correlations witnessed.

The graph shows the location of the cities analyzed.
